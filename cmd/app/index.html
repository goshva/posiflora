Как узнать chatId:
1. Напишите боту @userinfobot в Telegram → он пришлёт ваш личный chat_id
2. Для группы/канала добавьте @getidsbot или @myidbot → бот напишет id чата (обычно начинается с -100...)

После сохранения Chat ID будет замаскирован для безопасности.
